Utilizando diversas tags <img>

Em nossas páginas podemos ter uma ou mais imagens, e cada uma delas pode ter atributos diferentes, como width e height, para controlar seu tamanho e aparência.
Devemos procurar organizar as imagens de forma que elas se encaixem bem no layout da página, garantindo uma boa experiência para o usuário.

Porem, é importante lembrar que o uso excessivo de imagens pode afetar o desempenho da página, tornando-a mais lenta para carregar.
Por isso, é recomendado otimizar as imagens antes de inseri-las na página, reduzindo seu tamanho sem perder qualidade, e utilizando formatos adequados para a web, como JPEG, PNG ou WebP.

Outra regra importante é sempre organizar nossos arquivos de imagem de forma eficiente, mantendo-os em uma estrutura lógica e acessível.
Isso facilita a manutenção do site e garante que as imagens sejam carregadas corretamente.
Criar diretórios específicos para as imagens, como "imagens" ou "assets", é uma prática comum para manter a organização do projeto.
